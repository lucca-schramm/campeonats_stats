# Frontend - Campeonatos Stats

Frontend Vue.js com Tailwind CSS para visualização de estatísticas de futebol.

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

Acesse `http://localhost:3000`

## Build

```bash
npm run build
```

## Rotas

- `/brasileirao` - Visualização de estatísticas do Brasileirão
- `/chatbot` - Chatbot interativo para perguntas sobre futebol

## Tecnologias

- Vue 3
- Vue Router
- Pinia
- Tailwind CSS
- Axios
- Vite
